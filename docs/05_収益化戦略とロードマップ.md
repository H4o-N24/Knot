# 収益化戦略とロードマップ

## 料金プラン設計

| 機能 | Free | Pro | Team |
| :--- | :--- | :--- | :--- |
| **登録人数上限** | 10人 | 30人 | 100人以上 |
| **イベント作成** | 単純ランキング | 条件付き抽出 (必須メンバー等) | 高度なフィルタリング |
| **定員管理** | 基本機能のみ | 自動繰り上げ機能 | 詳細レポート出力 |
| **外部連携** | なし | なし | Google/Outlook同期 |
| **価格 (目安)** | ¥0 | ¥980/月 | ¥2,980/月 |

## 開発・API連携ロードマップ

### Phase 1: MVP (1ヶ月)
- Discord内での空き日登録・保存機能。
- 単純な人数ベースのランキング提案。
- `/event create` による基本イベント作成。

### Phase 2: Monetization (2ヶ月)
- Stripe連携によるサブスクリプション決済の実装。
- 条件付きフィルタリング（必須メンバー、曜日制限）の解放。
- 定員管理とキャンセル待ち自動繰り上げ。

### Phase 3: API Expansion (4ヶ月〜)
- **Google Calendar API 連携**: 予定の自動同期。
- **通知カスタマイズ**: DMリマインダーや外部Webhook連携。

### Phase 4: AI Optimization (6ヶ月〜)
- **OpenAI API 連携**: 自然言語によるイベント調整（「適当に来週のどっかで」への対応）。
- 過去の参加率に基づいた「成功確率スコア」の表示。
